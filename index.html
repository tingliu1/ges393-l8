HTML Basics

HTML documents are described by HTML tags. Each tag describes how different content will be displayed.

Tags are keywords (tag names) surrounded by < > :

<tagname> content </tagname>

Tags can also have attributes, which are extra bits of information. The example below defines a link (<a> tag) to the school homepage. The href attribute indicates the link's destination.

<a href = "http://www.neiu.edu/"> neiu homepage </a>

Bootstrap Templates - Best Practices

  Only change the index.html file for now: replace the content between the start and end tag. Take a quick look at the following HTML tags' usage before you start:

<!--...-->, <!DOCTYPE>, <a>, <b>, <br>, <button>, <div>, <h1>-<h6>, <head>, <html>, <i>, <iframe>, <li>, <link>, <nav>, <p>, <script>, <section>, <span>, <title>, <ul>

  You may also want to know the HTML class attribute.

  Make full use of the templates:
   - Copy existing components (e.g., sections, buttons, menus) to make a new one;
   - Hide unwanted components using comments (No rush to delete in case you need them later);

  You may replace the image files (same name and extension) in the img folder to have custom images.
